# 翻译来源

`translations/` 目录中的汉化词条主要来自社区项目 [OpenClawChineseTranslation](https://github.com/1186258278/OpenClawChineseTranslation)（MIT License），并按 [agentai2026/openclaw-zh](https://github.com/agentai2026/openclaw-zh) 的 CI 流程自动应用到上游 OpenClaw 源码。

- 约 **194** 个 JSON 模块、**3700+** 条替换规则
- 覆盖 Control UI（含设置页）、CLI、命令、配置 schema 等

使用或再分发时请保留 MIT 版权声明。
