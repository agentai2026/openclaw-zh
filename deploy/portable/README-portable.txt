OpenClaw 汉化版 — 便携包（内置 Node.js，无需单独安装 Node/npm）

快速开始：
  Windows：双击 bin\openclaw-gateway.cmd，或在 PowerShell 运行 bin\openclaw-gateway.ps1
  macOS / Linux：./bin/openclaw-gateway.sh onboard
              ./bin/openclaw-gateway.sh gateway run
  浏览器打开 http://127.0.0.1:18789
